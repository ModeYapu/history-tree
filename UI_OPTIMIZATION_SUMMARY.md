- 基础字号从 15px → 16px

- 输入框有明显的光晕效果
- 滑块尺寸增大 8.3%，更容易操作

### 4. 视觉反馈更加丰富
- 新增脉冲动画（加载文字）
- 新增浮动动画（空状态图标）
- 新增闪烁动画（骨架屏）
- 新增淡入淡出动画（交互提示）

### 5. 响应式布局优化
- 移动端详情卡片高度自适应
- 时间轴位置调整避免遮挡
