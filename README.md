# 🌳 历史之树 - Tree of History

一个交互式的中国与世界历史可视化Web应用，以树形结构展现人类历史进程。

**版本**: v2.0.0  
**更新日期**: 2026-02-27  

## 🎯 特性

### 核心功能
- ✅ **树形可视化** - 历史以树形结构展开，树干为时间轴，枝干为分类，树叶为具体事件/人物
- ✅ **交互式操作** - 点击展开、拖拽移动、滚轮缩放
- ✅ **详情展示** - 悬停预览、点击查看详细信息
- ✅ **搜索功能** - 快速查找历史事件和人物
- ✅ **筛选系统** - 按时期、类型、领域、重要程度筛选
- ✅ **时间轴导航** - 通过滑块快速跳转到不同历史时期

### v2.0 新增功能
- 🗺️ **地图视图** - 在地图上显示历史事件位置
- 🔗 **关系网络** - 可视化历史人物/事件关系
- 📥 **导出功能** - 支持PNG/SVG/JSON/PDF导出
- 🖼️ **图片支持** - 历史人物/事件图片展示

### 视觉效果
- 🎨 **精美设计** - 温暖的配色方案，营造历史氛围
- ✨ **流畅动画** - 树生长动画、节点发光、飘动效果
- 🌈 **颜色编码** - 不同领域用不同颜色标识
- 📱 **响应式设计** - 适配桌面和移动设备

## 📖 使用指南

### 基本操作

1. **展开节点**
   - 点击节点展开/折叠子节点
   - 圆形越大，层级越高

2. **移动视图**
   - 鼠标拖拽移动
   - 滚轮缩放
   - 双击放大

3. **查看详情**
   - 悬停查看预览
   - 点击查看完整信息
   - ESC关闭详情卡片

4. **搜索**
   - 在搜索框输入关键词
   - 支持事件名、人物名、描述搜索

5. **筛选**
   - 点击"筛选"按钮
   - 选择时期、类型、领域、重要程度
   - 点击"应用筛选"

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + F` | 聚焦搜索框 |
| `ESC` | 关闭详情卡片 |
| `+` | 放大 |
| `-` | 缩小 |
| `Home` | 重置视图 |

## 🎨 配色说明

### 节点颜色

| 颜色 | 含义 |
|------|------|
| 🔵 蓝色 | 科技 |
| 🔴 红色 | 政治 |
| 🟣 紫色 | 文化 |
| 🟢 绿色 | 经济 |
| 🟠 橙色 | 军事 |

### 节点大小

- **最大** - 时期节点（远古、古代、中世纪等）
- **中等** - 分支节点（文明起源、政治统一等）
- **最小** - 具体事件/人物

### 重要程度

- ⭐ - 一般重要
- ⭐⭐⭐ - 比较重要
- ⭐⭐⭐⭐⭐ - 非常重要
