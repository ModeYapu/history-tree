# 🌳 历史之树 - 使用指南

## 🚀 快速开始

### 1. 本地运行

```bash
# 进入项目目录
cd history-tree

# 启动本地服务器
python -m http.server 8000
# 或
npx http-server

# 访问
open http://localhost:8000
```

### 2. 基本操作

```
鼠标操作：
- 左键拖拽：旋转3D树
- 右键拖拽：平移视角
- 滚轮：缩放
- 点击树叶：查看历史事件详情
- 双击分支：展开/收起

键盘快捷键：
- Ctrl/Cmd + 1: 3D树视图
- Ctrl/Cmd + 2: 时间轴视图
- Ctrl/Cmd + 3: 网络视图
- Ctrl/Cmd + 4: 地图视图
- Ctrl/Cmd + 5: 卡片视图
- Ctrl/Cmd + K: 搜索
- Ctrl/Cmd + E: 导出
- Ctrl/Cmd + A: 数据分析
```

## 📚 功能说明

### 1. 🌲 3D历史之树

**核心视图** - 漂浮在空中的3D大树

```
结构：
- 主干：历史主脉络
- 分支：历史领域（政治/经济/文化/科技/军事）
- 树叶：历史事件/人物

特点：
- 真实的3D立体感
- 漂浮动画效果
- 光影系统
- 粒子效果（历史尘埃）
```

### 2. ⏱️ 时间轴视图

**时间维度** - 线性时间轴展示

```
功能：
- 时期块显示
- 事件标记
- 时间筛选
- 时期导航
```

### 3. 🕸️ 关系网络视图

**关联发现** - 力导向图展示关系

```
关系类型：
- 绿色：直接因果
- 蓝色：影响关系
- 橙色：跨时空关联
- 紫色：同期对比
```

### 4. 🗺️ 地图视图

**空间维度** - 地理分布展示

```
功能：
- 历史事件地理标记
- 分类颜色编码
- 弹窗详情
```

### 5. 🎴 卡片视图

**瀑布流** - 卡片式浏览

```
特点：
- 瀑布流布局
- 悬停效果
- 分类标签
- 快速浏览
```

## 🤖 AI功能

### 1. 智能关联发现

点击任意历史事件，AI会自动发现：

```
- 直接因果：导致该事件的原因
- 间接影响：该事件的影响
- 跨时空关联：不同时期的相似现象
- 同期对比：同时期其他文明的对比
```

### 2. 哲学思辨

深度分析历史事件：

```
6个哲学维度：
1. 本体论 - 存在的本质
2. 认识论 - 认识的可能性
3. 价值论 - 价值判断
4. 历史哲学 - 历史规律
5. 社会哲学 - 社会本质
6. 技术哲学 - 技术意义

多元视角：
- 决定论 vs 自由意志
- 必然性 vs 偶然性
- 普遍性 vs 特殊性
- 进步 vs 循环
```

### 3. 智能问答

提出任何历史问题，AI会：

```
1. 直接回答
2. 提供历史证据
```

## ❓ 常见问题

### Q1: 3D树加载慢怎么办？

```
A: 
1. 等待数据加载完成
2. 检查网络连接
3. 刷新页面重试
4. 使用其他视图模式
```

### Q2: AI响应慢怎么办？

```
A:
1. AI需要思考时间
2. 简化问题复杂度
3. 使用本地缓存
4. 检查API配置
```

### Q3: 如何添加新的历史数据？

```
A:
1. 编辑 data/historical-dataset.js
2. 按照格式添加
3. 刷新页面
```
