### MCP工具详细说明

#### 1. search_history
搜索历史事件、人物或时期

**参数**:
- `query` (string) - 搜索关键词
- `filters` (object) - 筛选条件
  - period - 历史时期
  - category - 类别
  - region - 地区
  - importance - 重要程度
- `limit` (number) - 返回数量

**返回**: 匹配的历史节点列表

#### 2. analyze_connections
AI分析事件关联关系

**参数**:
- `node_id` (string) - 节点ID
- `depth` (number) - 分析深度（1-3）
- `include_ai_analysis` (boolean) - 是否包含AI分析

**返回**: 关联关系列表（本地+AI发现）

#### 3. ask_history_question
向AI历史助手提问

**参数**:
- `question` (string) - 历史问题
- `context` (object) - 上下文
  - related_nodes - 相关节点
  - time_period - 时间范围

**返回**: AI回答 + 置信度 + 来源

#### 4. generate_timeline
生成历史时间线

**参数**:
- `theme` (string) - 主题
- `start_year` (number) - 起始年份
- `end_year` (number) - 结束年份
- `format` (string) - 格式（json/markdown/html）

**返回**: 格式化的时间线

#### 5. recommend_content
智能推荐内容

**参数**:
- `interests` (array) - 兴趣列表
- `current_node_id` (string) - 当前节点
- `limit` (number) - 推荐数量

**返回**: 推荐内容列表

#### 6. compare_events
对比历史事件

**参数**:
- `event_ids` (array) - 事件ID列表
- `aspects` (array) - 对比维度
  - cause - 原因
  - effect - 影响
  - significance - 意义
  - similarity - 相似点
  - difference - 不同点

**返回**: 对比分析结果

#### 7. trace_influence_chain
追踪影响链

**参数**:
- `node_id` (string) - 起始节点
- `direction` (string) - 方向（forward/backward/both）
- `max_depth` (number) - 最大深度

**返回**: 影响链树形结构
