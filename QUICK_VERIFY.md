# 🚀 快速验证指南

## 立即测试修复

### 1. 确保服务运行
```bash
# 检查端口
lsof -i :8203

# 如果没有运行，启动服务器
python3 -m http.server 8203
```

### 2. 打开浏览器
访问: **http://localhost:8203**

### 3. 检查控制台
按 `F12` 打开开发者工具，查看Console标签页。

**应该看到**:
```
✅ 核心修复补丁已加载
🌳 历史之树 v4.4 初始化中...
✅ 搜索引擎已初始化
✅ 数据已加载: XX 个节点
✅ 应用初始化完成
```

### 4. 测试功能清单

- [ ] **页面加载**: 加载屏幕消失，显示主界面
- [ ] **3D树**: 自动显示旋转的3D历史树
- [ ] **导航按钮**: 点击顶部按钮切换视图
  - [ ] 🌲 3D树
  - [ ] ⏱️ 时间轴
  - [ ] 🕸️ 网络
  - [ ] 🗺️ 地图
  - [ ] 🎴 卡片
- [ ] **搜索**: 按`Ctrl+K`或点击搜索框，输入"秦"或"唐朝"
- [ ] **AI聊天**: 点击右下角🤖按钮，打开聊天面板
- [ ] **快捷键**: `Ctrl+1-5`切换视图，`Ctrl+E`导出

### 5. 使用诊断页面
访问: **http://localhost:8203/test.html**

此页面会自动检查所有核心功能并显示诊断报告。

## 常见问题

**Q: 3D树不显示**
A: 检查浏览器是否支持WebGL，查看控制台错误

**Q: 搜索无结果**
A: 确保数据文件已加载，检查`HistoricalDataset`是否存在

**Q: AI聊天不响应**
A: 需要配置API密钥，查看`mcp-server/.env`
