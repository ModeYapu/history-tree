# 🌳 历史之树 - AI驱动的历史发现引擎

## 🎯 项目重新定位

**不是教育平台，而是历史发现与拓展引擎**

### 核心理念
```
传统历史学习：时间线 + 事件列表
我们的目标：发现隐藏的历史关联

利用大模型能力：
- 发掘事件背后的因果关系
- 找出不同时期、不同地域的隐藏联系
- 回答"为什么"类的历史问题
- 揭示历史发展的深层逻辑
```

---

## 🔍 核心功能

### 1. 智能关联发现

**功能描述**：
```
输入：历史事件/人物/时期
输出：AI分析的相关联事件 + 关联原因

示例：
Q: 五代十国时期才大规模刻经，为什么？
A: 发现关联：
   1. 佛教在唐末遭受打击，需要重建正统性
   2. 印刷术虽已发明，但技术成熟需要时间
   3. 政治分裂导致各地独立刻经
   4. 与同时期阿拉伯帝国的知识传播形成对比
```

**技术实现**：
- 大模型分析历史数据
- 多维度关联挖掘
- 因果推理
- 跨时空对比

---

### 2. 关系网络可视化

**核心视图**：
```
不只是时间线，而是：
- 因果关系网络
- 影响力流向图
- 跨时空连接
- 多维度关系
```

**可视化要素**：
```
节点：事件/人物/概念
边：关系类型
  - 直接因果 (实线)
  - 间接影响 (虚线)
  - 同期对比 (双向箭头)
  - 跨时空联系 (特殊标记)

权重：
  - 强弱关系（线粗细）
  - 时间跨度（线颜色）
  - 影响程度（节点大小）
```

---

### 3. 深度问答系统

**问题类型**：
```
1. 因果问题
   "为什么X导致了Y？"
   "X的背后原因是什么？"

2. 关联问题
   "X和Y有什么关系？"
   "还有哪些事件与X相关？"

3. 对比问题
   "X时期和Y时期有什么相似？"
   "为什么X地区和Y地区发展不同？"

4. 发现问题
   "X事件有哪些不为人知的影响？"
   "挖掘X背后的隐藏联系"
```

**回答格式**：
```
直接答案 + 证据链 + 可视化展示

示例：
Q: 为什么宋朝经济繁荣？

A:
直接原因：
1. 农业技术革新（占城稻引进）
2. 商业革命（纸币、市场网络）
3. 海上贸易（丝绸之路转移）

深层关联（AI发现）：
→ 与同时期欧洲商业革命对比
→ 与明朝海禁政策形成对比
→ 影响了后来的资本主义萌芽

[显示关系网络图]
```

## 🤖 大模型集成

### 模型选择
```
主模型：Claude 3.5 Sonnet (深度分析)
辅助模型：GPT-4 Turbo (快速检索)
本地模型：DeepSeek V3 (离线分析)
```

### Prompt设计
```
你是一个历史发现引擎，专门分析历史事件之间的深层关联。

你的任务是：
1. 发掘表面事件背后的因果链条
2. 找出不同时空的隐藏联系
3. 提供多维度的历史解释
4. 指出历史发展的规律和趋势

分析框架：
- 政治维度
- 经济维度
- 技术维度
- 文化维度
- 地理维度
- 国际对比

输出格式：
- 关联类型
- 关联强度
- 证据支持
- 可视化建议
```
