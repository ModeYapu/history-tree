## 🚀 使用方式

### 基本使用
```bash
cd /root/.openclaw/workspace/history-tree
./start.sh
# 访问 http://localhost:8000
```

### 切换视图
- 点击"视图模式"按钮
- 或使用右侧扩展按钮

### 导出
- 点击导出按钮 📥
- 选择格式（PNG/SVG/JSON/PDF）

### 地图视图
- 点击地图按钮 🗺️
- 查看历史事件地理分布
- 点击标记查看详情

### 关系网络
- 点击关系按钮 🔗
- 探索历史人物/事件关系
- 拖拽节点调整布局
