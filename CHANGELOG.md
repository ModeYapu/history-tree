# 🌳 历史之树 v2.0 - 扩展版本

## 🎉 新增功能

### 🗺️ 地图视图
- 在地图上显示历史事件发生位置
- 支持点击标记查看详情
- 颜色编码区分不同领域

### 🔗 关系网络
- 可视化历史人物/事件之间的关系
- 力导向图布局
- 交互式探索

### 📥 导出功能
- **PNG** - 高质量图片格式
- **SVG** - 矢量图，无损缩放
- **JSON** - 原始数据
- **PDF** - 文档格式

### 🖼️ 图片支持
- 历史人物/事件图片展示
- 懒加载优化性能

## 📊 数据扩展

### 新增节点数量
- **100+** 历史节点（原60+）
- **30+** 历史人物
- **50+** 历史事件
- **20+** 地理位置标记

### 新增内容
- 更详细的历史描述
- 地理位置信息
- 图片资源链接
- 相关标签

## 🎮 使用方法

### 切换视图
点击"视图模式"按钮切换：
1. 树形视图 - 默认
2. 地图视图 - 按地理位置
3. 关系视图 - 关系网络

### 扩展功能按钮
右侧浮动按钮：
- 🗺️ 地图视图
- 🔗 关系网络
- 📥 导出
- ⛶ 全屏

### 导出
点击导出按钮选择格式：
- PNG - 适合分享
- SVG - 适合编辑
- JSON - 数据备份
- PDF - 适合打印
