## 🚀 如何使用

### 快速开始
```bash
# 1. 进入项目目录
cd history-tree

# 2. 启动服务器
./start.sh

# 3. 打开浏览器
http://localhost:8000
```

### 基本操作
```
- 鼠标拖拽：旋转3D树
- 滚轮：缩放
- 点击树叶：查看详情
- Ctrl+K：搜索
- Ctrl+1-5：切换视图
```
